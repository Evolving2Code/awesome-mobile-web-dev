⚒️## Terminal & Linux Environments

*   [Proot-Distro](https://github.com/termux/proot-distro) - An official Termux utility to easily manage and run isolated Linux distributions (Ubuntu, Debian) inside Termux without root.
*   [Termux](https://termux.dev/) - A powerful terminal emulator and Linux environment for Android. No root required.
*   [Termux:API](https://github.com/termux/termux-api) - Expose device functionalities (like clipboard, camera, and storage) directly to the Termux command line.
*   [Zsh + Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh) - Enhance your Termux shell with autocomplete, syntax highlighting, and themes to make navigating directories on a small screen incredibly efficient.

🎨## Code Editors & IDEs

*   [Acode](https://acode.foxdebug.com/) - A powerful, lightweight text editor and IDE for Android with a fast, built-in HTML/server preview feature.
*   [Acode Git Extension](https://github.com/Acode-plugins) - A crucial plugin for Acode that allows you to stage, commit, and push code directly from the editor's UI without needing to hop back to the terminal.
*   [Acode Prettier Extension](https://github.com/Acode-plugins) - Keeps your code automatically formatted on save inside Acode, matching the standard formatting of desktop environments.
*   [code-server](https://github.com/coder/code-server) - Run VS Code locally inside Termux or on a remote server and access the full IDE UI via any mobile browser.
*   [Neovim](https://neovim.io/) - Vim-fork focused on extensibility and usability, highly optimized for blazing-fast terminal editing on phones.

🦾## AI & Productivity

*   [Claude Code](https://docs.anthropic.com/claude/docs/claude-code) - An agentic CLI tool by Anthropic for terminal-based coding, refactoring, and seamless git operations.
*   [GitHub CLI (gh)](https://cli.github.com/) - Bring GitHub pull requests, issues, and repository management straight to your Termux terminal.
*   [OpenRouter](https://openrouter.ai/) - A unified API to access a massive selection of LLMs, perfect for powering lightweight mobile terminal clients.

🌐## Database & Backend BaaS

*   [Prisma](https://www.prisma.io/) - Next-generation Node.js and TypeScript ORM. Works perfectly with Supabase and makes writing type-safe database queries seamless even when typing on a virtual keyboard.
*   [Supabase](https://supabase.com/) - An open-source Firebase alternative providing a full Postgres database, authentication, and instant API pipelines.
*   [Turso](https://turso.tech/) - An edge-hosted database based on libSQL/SQLite that is incredibly lightweight and perfect for mobile-first testing and prototyping.

📦## Package Managers & Runtimes

*   [Bun](https://bun.sh/) - A fast all-in-one JavaScript runtime, bundler, test runner, and package manager designed for speed.
*   [FnM (Fast Node Manager)](https://github.com/Schniz/fnm) - A blazing fast Node.js version manager built in Rust, perfect for quickly switching Node versions in a tight mobile terminal environment.
*   [Node.js](https://nodejs.org/) - Run server-side JavaScript, use npm/npx, and spin up local development servers directly on-device.
*   [pnpm](https://pnpm.io/) - Fast, disk space efficient package manager—crucial for managing limited storage space on mobile devices.

## Communities
