# Contributing to Awesome Mobile Web Dev

Contributions are what make the open-source community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

## Contribution Guidelines

1. **Quality over Quantity:** Ensure the tool, article, or resource is high quality and genuinely useful for mobile-first development.
2. **Formatting:** Match the existing formatting exactly: `* [Name](Link) - Description.`
3. **Alphabetical Order:** Please insert your link into the correct section in alphabetical order.
4. **No Self-Promotion Duplication:** If you are submitting your own project, ensure it has a clear, objective description.

## How to Contribute

1. Fork the Project.
2. Create your Feature Branch (`git checkout -b feature/AmazingResource`).
3. Commit your Changes (`git commit -m 'Add some AmazingResource'`).
4. Push to the Branch (`git push origin feature/AmazingResource`).
5. Open a Pull Request.
